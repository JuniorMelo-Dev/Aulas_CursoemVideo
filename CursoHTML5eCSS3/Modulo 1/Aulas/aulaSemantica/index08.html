Principais Formatações

Negrito / Destaque

Destacando termo em negrito usando a tag B. (não-semântico) HTML4

Destacando termo em negrito usando a tag STRONG. (semântica) HTML5

Itálico / Ênfase

Destacando termo em itálico usando a tag I.(não-semântico) HTML4

Destacando termo em itálico usando a tag EM. (semântico) HTML5

Marca Texto

Destacando marcação em texto usando a tag MARK. (semântica) HTML5

Textos Grande e Pequeno

Destacando texto grande e texto pequeno usando as tags BIG (não é mais usada) HTML4 e SMALL. (continua em uso) HTML5

Texto Deletado

Destacando texto deletado, mas presente no documento para ser observado, usando a tag DEL.

Texto Inserido

Destacando texto sublinhado usando a tag U. (não-semântica) HTML4

Destacando texto sublinhado usando a tag INS. (semântica) HTML5

Texto Sobrescrito e subscrito

Formatando textos como fórmulas matemáticas => 2x2+5, usando tag SUP.

Formatando textos como fórmulas de fisíca => H2O, usando tag SUB.
